# index-cn-notes

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的静态仓库。本仓库不针对任何特定域名或网站，仅作为普通 HTML 文档的集中存放与版本管理空间。每个 HTML 文件均为自包含的独立页面，可离线查看或部署到任意静态托管服务。

## 目录说明

```
index-cn-notes/
├── pages/          # 存放所有 HTML 页面文件
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/         # 页面所需的静态资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
├── README.md       # 本文件
└── LICENSE         # 许可证文件
```

- **pages/**：所有归档的 HTML 页面均存放于此目录下，文件名建议使用有意义的英文或拼音标识。
- **assets/**：存放页面引用的外部资源，按类型分目录管理，便于维护和复用。

## 页面归档说明

- 每个 HTML 页面均为独立完整文档，包含必要的 `<head>` 和 `<body>` 结构。
- 页面内容以中文为主，不涉及特定商业、营销或诱导性信息。
- 归档页面可能包含笔记、文档片段、工具页面或实验性内容，具体用途由文件名和内部注释说明。
- 页面之间无强依赖关系，可单独访问或部署。

## 维护说明

- 本仓库由普通维护者管理，欢迎提交 Issue 或 Pull Request 进行内容修正或新增页面。
- 新增页面请放入 `pages/` 目录，并确保文件名不与已有文件冲突。
- 若页面引用外部资源，请优先使用 `assets/` 目录下的本地资源，或确保外部链接长期有效。
- 本仓库不保证所有页面永久可用，但会尽力保持历史版本的完整性。
